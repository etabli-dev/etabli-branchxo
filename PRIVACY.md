# Privacy Policy — branchxo

**Effective date:** 2026-06-23
**App:** branchxo (Multiverse Tic-Tac-Toe)
**Publisher:** Raban Heller

## Short version

branchxo collects **no data**. The app does not phone home, does not include
any analytics SDK, does not include any ad SDK, does not include any crash
reporter. There is no network code in the app at all.

Game state (your move history and the multiverse tree) is saved **only on
your device** via the operating system's local storage (`AsyncStorage` on
iOS and Android). It is never transmitted anywhere. Uninstalling the app
deletes that local storage.

## Children

The app is a 3×3 board game with no chat, no user-generated text, no images,
no third-party content, and no network. It is suitable for all ages.
